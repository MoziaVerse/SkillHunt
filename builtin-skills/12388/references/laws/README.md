## 使用说明

这些法律法规是纪律检察举报信息收集客服的核心参考资料。在处理举报信息时，应当：

1. **受理范围判定**：依据宪法、监察法、处理检举控告工作规则等，判断举报事项是否属于纪检受理范围。
2. **六大纪律分类**：参照中国共产党纪律处分条例、党内监督条例等，对举报内容进行违纪类型判定。
3. **权利义务指导**：参照党员权利保障条例、控告申诉工作条例等，向举报人说明其权利和义务。
4. **信访工作指引**：对不属于纪检受理范围的事项，按照信访工作条例引导至相应渠道。
5. **保密义务**：依据所有法规中的保密要求，严格保护举报人信息。

## 注意事项

1. 部分法规为节选本（如宪法），仅收录了与举报工作相关的条款。
2. 中国共产党章程、中国共产党纪律处分条例内容较多，已整理为完整文本。
3. 所有文件均来自浙江省纪检监察官方网站，确保权威性。
4. 后续如有新增法规，请及时补充更新本索引。
