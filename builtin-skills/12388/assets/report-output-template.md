# 举报信息文件输出模板

> 用于生成最终保存的结构化文件。

## Markdown 版本 (report.md)

```markdown
# 举报信息登记

## 基本信息

- **采集时间**：{{采集日期时间}}
- **采集渠道**：{{渠道标识}}
- **举报方式**：{{实名/匿名}}

## 举报人信息

{{若实名}}
- 姓名：{{举报人姓名}}
- 联系电话：{{联系电话}}
- 联系地址：{{联系地址}}
- 与被举报人关系：{{关系 或 "未提供"}}
{{若匿名}}
- （匿名举报，举报人信息未采集）
{{/若}}

## 被举报人信息

- 姓名：{{被举报人姓名}}
- 单位/组织：{{所在单位}}
- 职务：{{职务 或 "未提供"}}
- 其他身份信息：{{其他 或 "未提供"}}

## 举报内容

### 违纪类型

{{违纪类型列表}}

### 事实经过

- **发生时间**：{{发生时间}}
- **发生地点**：{{发生地点}}
- **事实描述**：

{{事实经过描述}}

- **涉及金额**：{{涉及金额 或 "未提及"}}
- **其他涉及人员**：{{其他人员 或 "未提及"}}

## 证据材料

- **是否有证据**：{{是/否}}
- **证据类型**：{{类型 或 "无"}}
- **证据数量**：{{数量 或 "无"}}
- **证据说明**：{{说明 或 "无"}}

## 备注

{{补充说明 或 "无"}}
```
