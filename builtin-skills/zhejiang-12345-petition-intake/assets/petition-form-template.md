# 诉求记录模板

> 用于生成第一版保存记录。

## Markdown 版本 (<姓名或匿名>-<YYYY-MM-DD>.md)

```markdown
# 诉求记录

- **记录时间**：{{采集时间}}
- **类型**：{{服务类型}}
- **姓名**：{{诉求人姓名 或 "未提供"}}
- **手机号**：{{联系方式}}
- **联系地址**：{{联系地址 或 "未提供"}}
- **标题**：{{标题}}
- **涉及对象**：{{涉及对象 或 "未提供"}}
- **属地**：{{所在地区}}
- **事实经过**：

{{事实经过}}

- **核心诉求/建议**：{{核心诉求}}
- **附件情况**：{{附件情况 或 "未提供"}}
- **备注**：{{备注 或 "无"}}
```
