# 立案标准

## 一、立案基本条件

### 1.1 刑事案件立案条件

根据《公安机关办理刑事案件程序规定》，同时具备以下条件应当立案：

1. **有犯罪事实**
   - 有证据证明发生了犯罪行为
   - 有证据证明犯罪行为是嫌疑人实施的

2. **需要追究刑事责任**
   - 犯罪事实依照刑法规定构成犯罪
   - 未过追诉时效
   - 无豁免情形

3. **属于管辖范围**
   - 属于公安机关管辖的案件
   - 属于本地区域管辖范围

### 1.2 不予立案情形

| 情形类型 | 说明 |
|---------|------|
| 情节显著轻微 | 不认为是犯罪的 |
| 犯罪已过追诉时效 | 依据刑法规定不再追诉的 |
| 特赦免除刑罚 | 经特赦免除刑罚的 |
| 告诉才处理 | 未提出告诉的 |
| 嫌疑人死亡 | 嫌疑人死亡的 |
| 其他依法不追究 | 其他法律规定免予追究的 |

## 二、常见罪名立案标准

### 2.1 盗窃罪

- **数额较大**：1000-3000元以上（各地标准不同）
- **多次盗窃**：2年内3次以上
- **入户盗窃**：非法进入供他人家庭生活
- **携带凶器盗窃**：携带枪支、管制刀具等
- **扒窃**：在公共场所窃取他人财物

### 2.2 诈骗罪

- **数额较大**：3000-10000元以上
- **电信网络诈骗**：3000元以上即可立案

### 2.3 故意伤害罪

- **轻伤**：造成轻伤以上后果
- **情节恶劣**：持械、多人等加重情节

### 2.4 抢劫罪

- 以暴力、威胁方法抢劫公私财物
- 无论数额大小均应立案

### 2.5 抢夺罪

- **数额较大**：1000-3000元以上
- **情节严重**：多次、凶器等加重情节

## 三、立案程序

### 3.1 受案

```
接警 → 制作受案登记表 → 出具受案回执
```

### 3.2 审查

- 初步核查证据材料
- 判断是否构成犯罪
- 确定管辖权

### 3.3 立案决定

- 符合条件的，制作《立案决定书》
- 不符合条件的，制作《不予立案通知书》并说明理由

### 3.4 告知与通知

- 立案后通知报案人、控告人
- 不予立案的，说明理由并告知复议权利

## 四、行政案件受案

### 4.1 受案条件

- 有违反治安管理行为发生
- 需要给予治安管理处罚
- 属于公安机关管辖

### 4.2 与刑事案件的区分

| 刑事案件 | 行政案件 |
|---------|---------|
| 追究刑事责任 | 给予治安处罚 |
| 检察院审查起诉 | 公安机关直接处理 |
| 可能判处刑罚 | 拘留、罚款等 |
| 立案标准高 | 违法即处理 |

## 五、监督与救济

- **检察院监督**：检察院可要求说明不立案理由
- **复议复核**：控告人可申请复议
- **自诉途径**：被害人可直接向法院起诉
