# 接处警流程规范

## 一、接警规范

### 1.1 接警基本要求

- **24小时值班制度**：公安机关实行24小时接警值班制度
- **接警登记**：对接报的警情必须如实登记，内容包括：
  - 报警人姓名、联系电话
  - 发案时间、地点
  - 简要案情
  - 警情类别
- **紧急程度判断**：迅速判断警情性质和紧急程度

### 1.2 警情分类

| 警情类型 | 处置时限 | 备注 |
|---------|---------|------|
| 重大刑事案件 | 立即出警 | 杀人、抢劫、绑架等 |
| 重大治安案件 | 立即出警 | 群体性事件、械斗等 |
| 一般刑事案件 | 15分钟内 | 盗窃、诈骗等 |
| 一般治安案件 | 30分钟内 | 纠纷、求助等 |
| 非紧急求助 | 60分钟内 | 咨询、服务类 |

## 二、处警规范

### 2.1 处警力量配置

- **刑事案件**：不少于2名民警，携带必要警械
- **群体性事件**：根据规模增派警力
- **暴力犯罪**：配备防暴装备，请求支援

### 2.2 现场处置要点

1. **迅速控制现场**
   - 隔离围观群众
   - 控制嫌疑人
   - 保护现场证据

2. **伤员救助**
   - 优先抢救受伤人员
   - 及时拨打120

3. **现场调查**
   - 寻找目击证人
   - 收集初步证据
   - 制作现场笔录

## 三、警情反馈与移交

### 3.1 反馈要求

- 到达现场后及时反馈
- 处置过程中持续报告
- 处置完毕后书面报告

### 3.2 案件移交

- 行政案件：移交派出所处理
- 刑事案件：移交刑侦部门
- 重大案件：立即上报

## 四、注意事项

- 严禁推诿、拖延出警
- 严禁泄露报警人信息
- 注意执法安全，佩戴执法记录仪
- 规范用语，文明执法
