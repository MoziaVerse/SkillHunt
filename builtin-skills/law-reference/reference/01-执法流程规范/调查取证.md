# 调查取证

## 一、取证基本原则

### 1.1 合法性原则

- 主体合法：具有执法资格的人员
- 程序合法：遵守法定程序
- 手段合法：严禁刑讯逼供、诱供

### 1.2 客观性原则

- 全面收集证据
- 不偏不倚
- 如实记录

### 1.3 关联性原则

- 证据与案件事实相关
- 能够证明案件事实

## 二、常见证据类型

### 2.1 物证

| 类型 | 收集要点 | 保管要求 |
|------|---------|---------|
| 凶器 | 记录特征、提取指纹 | 单独封装 |
| 赃物 | 拍照固定、登记清单 | 妥善保管 |
| 书证 | 原件优先、复印件注明 | 防止损坏 |

### 2.2 视听资料

- **监控录像**：及时调取、刻录保存
- **录音录像**：注明来源、制作人
- **电子数据**：依法提取、完整固定

### 2.3 证人证言

- **询问对象**：知道案件情况的人
- **询问要求**：个别进行、如实记录
- **权利告知**：告知如实作证义务

### 2.4 被害人陈述

- 及时询问、固定陈述
- 注意情绪安抚
- 制作询问笔录

### 2.5 嫌疑人供述

- 禁止刑讯逼供
- 如实记录、嫌疑人核对签字
- 允许辩解

### 2.6 鉴定意见

- 指派或聘请有资质的鉴定人
- 告知鉴定意见
- 可申请补充鉴定或重新鉴定

## 三、取证程序要求

### 3.1 现场勘查

```
保护现场 → 拍照录像 → 绘制现场图 → 提取痕迹物证 → 制作勘查笔录
```

### 3.2 搜查

- **搜查证**：除紧急情况外需持搜查证
- **见证人**：应当有见证人在场
- **笔录**：制作搜查笔录
- **清单**：开具扣押物品清单

### 3.3 扣押

- **范围**：与案件有关的物品
- **手续**：开具扣押清单
- **保管**：妥善保管、随案移送

### 3.4 查询冻结

- **存款汇款**：可查询冻结嫌疑人存款
- **期限**：冻结不超过6个月
- **续冻**：可续冻，每次不超过6个月

## 四、证据审查要点

### 4.1 真实性审查

- 来源是否可靠
- 内容是否真实
- 有无伪造变造

### 4.2 合法性审查

- 收集程序是否合法
- 手段是否合法
- 形式是否合法

### 4.3 关联性审查

- 与案件事实是否相关
- 证明力大小
- 能否形成证据链

## 五、常见案件取证要点

### 5.1 盗窃案件

- 现场勘查（脚印、指纹）
- 赃物追缴
- 视频监控调取
- 销赃渠道调查

### 5.2 诈骗案件

- 资金流向追踪
- 通信记录调取
- 虚假身份查证
- 受害人询问

### 5.3 故意伤害案件

- 伤情鉴定
- 凶器提取
- 目击证人
- 现场监控

### 5.4 网络犯罪

- 电子数据提取
- IP地址追踪
- 账号注册信息
- 交易记录

## 六、注意事项

- 证据应当随案移送
- 不得隐匿、毁弃证据
- 非法证据予以排除
- 形成完整证据链
