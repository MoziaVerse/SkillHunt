# 公安业务知识库文档索引

## 文档库说明

本知识库为公安执法办案提供规范指引和知识支撑，内容涵盖执法流程、法律法规、文书模板、典型案例和常见问题解答。

## 📁 目录结构

### 01-执法流程规范
规范公安执法办案各环节的程序要求

- **接处警流程** - 接警、处警、现场处置规范
- **立案标准** - 立案条件、受案程序、不予立案情形
- **调查取证** - 证据收集、固定、审查要求
- **案件办理程序** - 刑事、行政案件办理全流程

### 02-法律法规条文
公安执法相关的法律法规条文

- **刑法相关** - 常见罪名、构成要件、量刑标准
- **刑诉法相关** - 刑事诉讼程序、强制措施、时限要求
- **治安管理处罚法** - 违法行为认定、处罚种类、程序规定

### 03-执法文书模板
各类执法文书的制作规范和模板

- **刑事法律文书** - 立案决定书、提请批准逮捕书、起诉意见书等
- **行政法律文书** - 受案登记表、行政处罚决定书、询问笔录等

### 04-典型案例
执法实践中的典型案例分析

- **刑事案件** - 常见刑事犯罪案例
- **行政案件** - 常见治安案件案例

### 05-常见问题指引
执法办案中的疑难问题解答

- **执法疑难问题** - 执法实践中的常见问题
- **程序操作指引** - 具体程序的操作要点
