# 行政法律文书模板

## 一、受案阶段

### 1.1 受案登记表

**要素：**
- 受案单位
- 受案时间
- 案件来源
- 报案人信息
- 违法嫌疑人信息（如有）
- 案件简要情况
- 受案意见
- 审批意见

### 1.2 受案回执

**内容：**
- 报案人信息
- 案件编号
- 受案时间
- 承办单位、联系人
- 注意事项

## 二、调查阶段

### 2.1 传唤证

**要素：**
- 被传唤人基本情况
- 传唤时间、地点
- 传唤理由
- 无正当理由不到场的后果
- 经办人、批准人
- 送达日期

### 2.2 询问笔录

**格式：**
- 时间、地点
- 询问人、记录人
- 被询问人基本情况（核实身份）
- 权利义务告知
- 问答内容
- 核对确认（"以上记录我看过，与我所说相符"）
- 签名捺印

**注意事项：**
- 单独询问
- 如实记录
- 不得诱导、威胁
- 被询问人核对后签名

### 2.3 勘验/检查笔录

**要素：**
- 时间、地点、天气
- 勘验/检查人员
- 见证人
- 过程描述
- 发现情况
- 绘图、照片
- 签名

### 2.4 扣押清单

**要素：**
- 编号
- 物品名称、数量、特征
- 扣押时间、地点
- 持有人/见证人
- 经办人
- 签名确认

## 三、决定阶段

### 3.1 公安行政处罚决定书

**结构：**

1. **首部**
   - 文书编号
   - 被处罚人信息
   - 违法事实

2. **正文**
   - 违法事实经过
   - 证据列举
   - 从重/从轻/减轻情节
   - 法律依据
   - 处罚决定

3. **尾部**
   - 履行方式和期限
   - 救济途径（复议、诉讼）
   - 作出机关
   - 日期、盖章

**示例：**
```
XX公安局公安行政处罚决定书
X公（行）决字〔2026〕第XXX号

被处罚人：张三，男，XXXX年X月X日出生，汉族，住址...

现查明：XXXX年X月X日X时许，张三在XX地点...（描述违法事实）

以上事实有：1.当事人陈述；2.证人证言；3.视听资料等证据证实。

根据《中华人民共和国治安管理处罚法》第X条之规定，现决定对张三处以行政拘留X日。

执行方式和期限：...
被处罚人不服本决定的，可在收到本决定书之日起六十日内向XX机关申请行政复议，或在六个月内向XX法院提起行政诉讼。

XX公安局
XXXX年X月X日
```

### 3.2 治安调解协议书

**要素：**
- 时间、地点
- 调解人员
- 当事人基本情况
- 纠纷事实经过
- 调解结果
- 履行方式和期限
- 当事人签名
- 调解人签名
- 日期

### 3.3 不予行政处罚决定书

**要素：**
- 当事人信息
- 案件情况
- 不予处罚理由
- 法律依据
- 日期、盖章

### 3.4 收容教育/强制隔离戒毒决定书

**要素：**
- 决定对象
- 违法事实
- 法律依据
- 决定内容
- 执行地点
- 救济途径
- 日期、盖章

## 四、执行阶段

### 4.1 行政拘留执行通知书

**要素：**
- 被拘留人信息
- 拘留期限
- 执行场所
- 送达日期

### 4.2 缴纳罚款通知书

**要素：**
- 缴款人
- 罚款金额
- 缴纳期限（15日内）
- 缴纳方式
- 逾期后果（加处罚款）

## 五、文书制作规范

### 5.1 基本要求

| 要求 | 说明 |
|------|------|
| 内容真实 | 如实反映情况 |
| 格式规范 | 按统一格式制作 |
| 语言准确 | 用词严谨，避免歧义 |
| 手续完备 | 签名、盖章、日期齐全 |
| 字迹清晰 | 工整书写，不得潦草 |

### 5.2 事实叙述

**要点：**
- 时间、地点、人物、事件、原因、结果
- 按时间顺序
- 客观描述
- 避免主观评价

### 5.3 法律引用

**规范：**
- 引用完整法律名称
- 具体到条、款、项
- 最新版本
- 格式：《中华人民共和国治安管理处罚法》第四十三条第一款

### 5.4 送达规范

**要求：**
- 直接送达首选
- 受送达人签收
- 拒绝签收的，可以留置送达
- 送达回证签名

## 六、常见问题

### 6.1 易错点

- 法律条款引用错误
- 事实叙述不清
- 手续不齐全
- 送达不规范
- 日期错误

### 6.2 注意事项

1. **时限意识**：各项决定都有时限要求
2. **程序意识**：严格按照法定程序
3. **证据意识**：所有决定要有证据支持
4. **权利保障**：告知当事人权利
5. **文书留存**：原件存档，复印件送达

### 6.3 文书修改

- 错误处划改，由修改人签名或盖章
- 不得涂改、刮擦
- 修改过多应重新制作
