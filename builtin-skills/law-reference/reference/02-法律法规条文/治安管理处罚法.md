# 治安管理处罚法

## 一、处罚种类

### 1.1 处罚种类（第10条）

| 种类 | 说明 |
|------|------|
| 警告 | 最轻处罚，书面告诫 |
| 罚款 | 一般200-500元，最高不超过5000元 |
| 行政拘留 | 1-15日，合并执行不超过20日 |
| 吊销公安机关发放的许可证 | 暂扣或吊销 |
| 限期出境/驱逐出境 | 适用于外国人 |

### 1.2 对未成年人的特殊规定

| 年龄 | 处罚规定 |
|------|---------|
| 不满14周岁 | 不予处罚，责令监护人管教 |
| 14-16周岁 | 不执行行政拘留 |
| 16-18周岁，初次违反 | 不执行行政拘留 |

## 三、办案程序

### 3.1 管辖

- **地域管辖**：由违法行为地公安机关管辖
- **级别管辖**：一般由派出所管辖，重大案件由县/市公安机关管辖

### 3.2 受案与调查

**受案：**
- 接报案制作《受案登记表》
- 出具《受案回执》

**调查措施：**
- 传唤嫌疑人（需传唤证，现场发现可口头传唤）
- 询问查证（一般不超过8小时，不超过24小时）
- 勘验、检查
- 鉴定、检测

### 3.3 证据收集

**证据类型：**
- 物证、书证
- 视听资料
- 电子数据
- 证人证言
- 被害人陈述
- 违法嫌疑人的陈述和申辩
- 鉴定意见
- 勘验、检查笔录

**要求：**
- 不得刑讯逼供
- 采用非法手段收集的证据应当予以排除

### 3.4 决定与执行

**处罚决定：**
- 调查终结后作出决定
- 制作《行政处罚决定书》
- 向被处罚人宣告并送达

**执行：**
- 罚款应在15日内缴纳
- 行政拘留送拘留所执行
- 拒不执行可强制执行

### 3.5 听证程序

**适用情形：**
- 吊销许可证
- 较大数额罚款（个人2000元以上）
- 责令停产停业

**程序：**
- 告知听证权利
- 当事人申请（3日内）
- 组织听证（7日前通知）

## 四、调解处理

### 4.1 可调解案件（第9条）

- 因民间纠纷引起的违反治安管理行为
- 情节较轻

**范围：**
- 殴打他人/故意伤害
- 故意损毁财物
- 诽谤、诬告陷害
- 干扰他人正常生活
- 侵犯隐私

### 4.2 调解程序

```
申请调解 → 调查了解 → 组织调解 → 达成协议 → 履行协议 → 结案
```

**结果：**
- 调解达成协议并履行的，不予处罚
- 调解未达成协议或达成协议不履行的，依法处罚

## 五、救济途径

### 5.1 行政复议

- **申请期限**：60日
- **复议机关**：上一级公安机关或本级人民政府

### 5.2 行政诉讼

- **起诉期限**：6个月（直接起诉）或15日（复议后起诉）
- **管辖法院**：行政机关所在地基层法院

## 六、与刑事案件的界限

### 6.1 区分标准

| 要素 | 治安案件 | 刑事案件 |
|------|---------|---------|
| 社会危害性 | 较小 | 较大 |
| 情节 | 较轻 | 较重或严重 |
| 侵犯法益 | 一般社会关系 | 重要社会关系 |
| 处罚性质 | 行政处罚 | 刑罚 |

### 6.2 常见罪名与违法行为对应

| 刑事罪名 | 违法行为 |
|---------|---------|
| 故意伤害罪 | 殴打他人、故意伤害 |
| 强奸罪 | 猥亵 |
| 强制猥亵、侮辱罪 | 猥亵 |
| 非法拘禁罪 | 非法限制人身自由 |
| 盗窃罪 | 盗窃 |
| 诈骗罪 | 诈骗 |
| 抢劫罪 | 抢夺（部分） |
| 寻衅滋事罪 | 扰乱公共秩序 |
| 妨害公务罪 | 阻碍执行职务 |

### 6.3 刑民交叉、刑行交叉处理

**原则：**
- 刑事优先：涉嫌犯罪的，先刑事程序
- 分别处理：刑事、行政、民事可分别处理
- 衔接配合：行政处罚与刑事处罚可折抵
