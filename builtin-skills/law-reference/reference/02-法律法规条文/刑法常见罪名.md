# 刑法常见罪名

## 一、侵犯财产罪

### 1.1 盗窃罪（第264条）

**构成要件：**
- 客体：公私财物所有权
- 客观方面：秘密窃取数额较大或多次盗窃
- 主体：一般主体（16周岁以上）
- 主观方面：直接故意，以非法占有为目的

**量刑标准：**

| 情节 | 量刑 |
|------|------|
| 数额较大/多次盗窃 | 3年以下有期徒刑、拘役或管制 |
| 数额巨大/其他严重情节 | 3-10年有期徒刑 |
| 数额特别巨大/其他特别严重情节 | 10年以上有期徒刑或无期徒刑 |

**数额标准参考：**
- 数额较大：1000-3000元
- 数额巨大：3-10万元
- 数额特别巨大：30-50万元

### 1.2 诈骗罪（第266条）

**构成要件：**
- 客体：公私财物所有权
- 客观方面：虚构事实或隐瞒真相，骗取数额较大财物
- 主观方面：直接故意，以非法占有为目的

**量刑标准：**

| 情节 | 量刑 |
|------|------|
| 数额较大 | 3年以下有期徒刑、拘役或管制 |
| 数额巨大/其他严重情节 | 3-10年有期徒刑 |
| 数额特别巨大/其他特别严重情节 | 10年以上有期徒刑或无期徒刑 |

### 1.3 抢劫罪（第263条）

**构成要件：**
- 客体：复杂客体（人身权+财产权）
- 客观方面：以暴力、威胁或其他方法劫取财物
- 主体：一般主体（14周岁以上）
- 主观方面：直接故意

**基本量刑：** 3年以上10年以下有期徒刑

**加重情节（10年以上、无期徒刑或死刑）：**
- 入户抢劫
- 在公共交通工具上抢劫
- 抢劫银行或其他金融机构
- 多次抢劫或抢劫数额巨大
- 致人重伤、死亡
- 冒充军警人员抢劫
- 持枪抢劫
- 抢劫军用物资或抢险、救灾、救济物资

### 1.4 抢夺罪（第267条）

**构成要件：**
- 客观方面：公然夺取数额较大财物
- 主观方面：直接故意，以非法占有为目的

**量刑标准：**

| 情节 | 量刑 |
|------|------|
| 数额较大 | 3年以下有期徒刑、拘役或管制 |
| 数额巨大/其他严重情节 | 3-10年有期徒刑 |
| 数额特别巨大/其他特别严重情节 | 10年以上有期徒刑或无期徒刑 |

### 1.5 敲诈勒索罪（第274条）

**构成要件：**
- 客观方面：威胁、要挟，索取数额较大财物
- 主观方面：直接故意

**量刑标准：**

| 情节 | 量刑 |
|------|------|
| 数额较大/多次敲诈 | 3年以下有期徒刑、拘役或管制 |
| 数额巨大/其他严重情节 | 3-10年有期徒刑 |
| 数额特别巨大/其他特别严重情节 | 10年以上有期徒刑 |

## 二、侵犯公民人身权利罪

### 2.1 故意伤害罪（第234条）

**构成要件：**
- 客体：他人身体健康权
- 客观方面：非法损害他人身体健康

**量刑标准：**

| 伤害程度 | 量刑 |
|---------|------|
| 轻伤 | 3年以下有期徒刑、拘役或管制 |
| 重伤 | 3-10年有期徒刑 |
| 死亡/以特别残忍手段致重伤造成严重残疾 | 10年以上有期徒刑、无期徒刑或死刑 |

**伤情鉴定标准：**
- 轻伤：造成肢体或容貌损害，听觉、视觉或其他器官功能部分障碍
- 重伤：使人肢体残废、毁人容貌、丧失听觉、视觉或其他器官功能

### 2.2 非法拘禁罪（第238条）

**构成要件：**
- 客观方面：非法剥夺他人人身自由
- 主观方面：故意

**量刑标准：**
- 基本刑：3年以下有期徒刑、拘役、管制或剥夺政治权利
- 致人重伤：3年以上10年以下
- 致人死亡：10年以上有期徒刑

**注意：** 使用暴力致人伤残、死亡的，按故意伤害罪、故意杀人罪定罪处罚。

### 2.3 寻衅滋事罪（第293条）

**行为方式：**
- 随意殴打他人，情节恶劣
- 追逐、拦截、辱骂、恐吓他人，情节恶劣
- 强拿硬要或任意损毁、占用公私财物，情节严重
- 在公共场所起哄闹事，造成公共场所秩序严重混乱

**量刑：** 5年以下有期徒刑、拘役或管制

**纠集他人多次实施，严重破坏社会秩序的：** 5-10年有期徒刑

## 三、妨害社会管理秩序罪

### 3.1 妨害公务罪（第277条）

**构成要件：**
- 客观方面：以暴力、威胁方法阻碍国家机关工作人员依法执行职务

**量刑：** 3年以下有期徒刑、拘役、管制或罚金

### 3.2 赌博罪（第303条）

**构成要件：**
- 客观方面：以营利为目的，聚众赌博或以赌博为业

**量刑：** 3年以下有期徒刑、拘役或管制，并处罚金

### 3.3 毒品犯罪

| 罪名 | 量刑 |
|------|------|
| 走私、贩卖、运输、制造毒品（第347条） | 根据毒品数量和情节，最高可判死刑 |
| 非法持有毒品（第348条） | 数量大的处7年以上有期徒刑 |
| 容留他人吸毒（第354条） | 3年以下有期徒刑、拘役或管制 |

## 四、常见罪名辨析

### 4.1 抢劫与抢夺

| 要点 | 抢劫 | 抢夺 |
|------|------|------|
| 暴力程度 | 抑制反抗 | 未达抑制程度 |
| 侵犯客体 | 财产权+人身权 | 仅财产权 |
| 主观内容 | 取财+人身侵害 | 仅取财 |

### 4.2 诈骗与盗窃

| 要点 | 诈骗 | 盗窃 |
|------|------|------|
| 取财方式 | 被害人自愿交付 | 秘密窃取 |
| 被害人认知 | 认识错误交付 | 不知情 |
| 行为人手段 | 欺骗 | 秘密 |
