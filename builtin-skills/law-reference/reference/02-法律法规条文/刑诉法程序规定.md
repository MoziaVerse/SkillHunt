# 刑事诉讼法程序规定

## 一、强制措施

### 1.1 拘传

**适用对象：** 未被羁押的犯罪嫌疑人、被告人

**时间限制：**
- 不得超过12小时
- 案情重大、复杂，不得超过24小时

**不得以连续拘传方式变相拘禁**

### 1.2 取保候审

**适用条件（第67条）：**

| 可以取保候审 | 应当取保候审 |
|------------|------------|
| 可能判处管制、拘役 | 可能判处管制、拘役 |
| 可能判处徒刑以上，不致发生社会危险性 | 患有严重疾病、生活不能自理 |
| 患有严重疾病、生活不能自理 | 怀孕或正在哺乳婴儿 |
| 怀孕或正在哺乳 | 生活不能自理，本人无社会危险性 |
| 羁押期限届满，案件未办结 | 羁押期限届满 |

**保证方式：**
- **保证金**：起点1000元
- **保证人**：符合条件的人

**期限：** 不得超过12个月

**被取保候审人义务（第69条）：**
- 未经执行机关批准不得离开居住的市、县
- 住址、工作单位、联系方式变更需24小时内报告
- 传讯时及时到案
- 不得以任何形式干扰证人作证
- 不得毁灭、伪造证据或串供

### 1.3 监视居住

**适用条件（第74条）：**
- 符合逮捕条件，但有特殊情形：
  - 患有严重疾病、生活不能自理
  - 怀孕或正在哺乳
  - 系生活不能自理的人的唯一扶养人
  - 案件特殊情况，需要监视居住

**期限：** 不得超过6个月

**执行方式：**
- 原则上在住处执行
- 无固定住处的，在指定居所执行
- 危害国家安全犯罪、恐怖活动犯罪，在住处执行可能有碍侦查的，经批准可指定居所

### 1.4 拘留

**适用对象（第80条）：**

| 情形 | 期限 |
|------|------|
| 正在预备犯罪、实行犯罪或在犯罪后即时被发觉 | 一般14日 |
| 被害人或在场亲眼看见的人指认他犯罪 | 流窜作案、多次作案、结伙作案的重大嫌疑分子：37日 |
| 在身边或住处发现犯罪证据 | |
| 犯罪后企图自杀、逃跑或在逃 | |
| 有毁灭、伪造证据或串供可能 | |
| 不讲真实姓名、住址，身份不明 | |
| 有流窜作案、多次作案、结伙作案重大嫌疑 | |

### 1.5 逮捕

**适用条件（第79条）：**

1. **有证据证明有犯罪事实**
2. **可能判处徒刑以上刑罚**
3. **采取取保候审尚不足以防止社会危险性**

**社会危险性情形：**
- 可能实施新犯罪
- 有危害国家安全、公共安全或社会秩序的现实危险
- 可能毁灭、伪造证据，干扰证人作证或串供
- 可能对被害人、举报人、控告人实施打击报复
- 企图自杀或逃跑

**审查批准：**
- 公安机关提请
- 检察院审查批准（7日内作出决定）
- 公安机关执行

**通知义务：**
- 逮捕后24小时内通知家属
- 无法通知的除外

### 1.6 强制措施期限

| 措施 | 最长期限 |
|------|---------|
| 拘传 | 24小时 |
| 取保候审 | 12个月 |
| 监视居住 | 6个月 |
| 拘留 | 一般14日，特定情况37日 |
| 羁押（逮捕后） | 二审前一般不超过12个月 |

## 二、侦查程序

### 2.1 讯问犯罪嫌疑人

**时间限制：**
- 传唤、拘传持续不得超过12小时
- 案情重大复杂，不得超过24小时

**权利保障：**
- 应当告知犯罪嫌疑人诉讼权利
-如实供述可从宽处理
- 对与本案无关的问题，有拒绝回答的权利

**未成年人：**
- 应当通知法定代理人或其他合适成年人到场
- 女性未成年犯罪嫌疑人，应当有女工作人员在场

**同步录音录像：**
- 重大犯罪案件应当录音录像

### 2.2 询问证人、被害人

**要求：**
- 个别进行
- 告知如实作证的义务
- 证人、被害人有权核对笔录

**保护措施：**
- 公开本人信息可能危及人身安全的，应当不公开真实姓名、住址
- 可以采取不暴露外貌、真实声音的出庭作证措施

### 2.3 勘验、检查

**主体：** 侦查人员

**见证人：** 应当有见证人在场

**检查身体：**
- 犯罪嫌疑人：如果拒绝，可以强制检查
- 被害人：应当自愿，不得强制

**人身检查特殊规定：**
- 检查妇女身体，应当由女工作人员或医师进行

### 2.4 搜查

**程序要求：**
- 必须持搜查证
- 紧急情况可例外（执行逮捕、拘留时遇紧急情况）

**要求：**
- 应当有被搜查人或其他见证人在场
- 制作搜查笔录
- 开具扣押物品清单

### 2.5 查询、冻结

**对象：** 存款、汇款、债券、股票、基金份额等财产

**程序：**
- 制作协助查询/冻结财产通知书
- 通知金融机构等单位执行

**期限：**
- 冻结存款、汇款等财产的期限为6个月
- 每次续冻期限不超过6个月

## 三、办案期限

### 3.1 侦查羁押期限

| 阶段 | 期限 | 备注 |
|------|------|------|
| 逮捕后侦查羁押 | ≤2个月 | 可延长1个月 |
| 案情复杂、期限届满不能终结 | +1个月 | 上一级检察院批准 |
| 交通不便、重大犯罪、流窜作案、涉及面广 | +2个月 | 省级检察院批准 |
| 可能判10年以上有期徒刑 | +2个月 | 省级检察院批准 |
| 特殊原因，长期不宜交付审判 | 再延长 | 最高检报全国人大批准 |

### 3.2 审查起诉期限

- **一般案件：** 1个月内作出决定，可延长15日
- **重大、复杂案件：** 可再延长15日

### 3.3 一审期限

- **公诉案件：** 2个月内宣判，至迟不超过3个月
- **可延长情形：** 死刑、附带民事诉讼、交通不便等，经批准可延长3个月

## 四、辩护与代理

### 4.1 辩护人权利

- 与在押犯罪嫌疑人、被告人会见、通信
- 查阅案卷材料
- 调取证据
- 提出意见

### 4.2 法律援助

**适用情形：**
- 犯罪嫌疑人、被告人因经济困难或其他原因未委托辩护人
- 盲、聋、哑人，未成年人，可能被判处无期徒刑、死刑的人

## 五、特别程序

### 5.1 未成年人刑事案件

- **原则：** 教育、感化、挽救
- **办案要求：** 专门机构或专人办理
- **封存记录：** 犯罪时不满18周岁，判5年以下有期徒刑的，封存犯罪记录

### 5.2 当事人和解的公诉案件

**适用范围：**
- 因民间纠纷引起的，涉嫌刑法分则第四章、第五章规定的犯罪案件，可能判处3年有期徒刑以下刑罚的
- 除渎职犯罪以外的可能判处7年有期徒刑以下刑罚的过失犯罪案件

**不适用：**
- 盗窃、诈骗等职务犯罪
- 严重侵害社会公共利益
- 严重危害公共安全

### 5.3 犯罪嫌疑人、被告人逃匿、死亡案件违法所得没收程序

**条件：**
- 贪污贿赂犯罪、恐怖活动犯罪等重大犯罪
- 通缉1年后不能到案
- 依法应当追缴违法所得

### 5.4 依法不负刑事责任的精神病人的强制医疗程序

**条件：**
- 实施暴力行为，危害公共安全或严重危害公民人身安全
- 经法定程序鉴定依法不负刑事责任
- 有继续危害社会可能
