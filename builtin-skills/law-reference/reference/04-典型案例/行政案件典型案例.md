# 行政案件典型案例

## 一、殴打他人案例

### 案例1：邻里纠纷殴打他人

**基本案情：**
2025年3月，张三与李四因楼道堆物问题发生争执。张三推搡李四，并用拳头击打李四面部，致李四面部软组织挫伤。经医院检查，李四面部多处软组织损伤，不构成轻微伤。

**调查情况：**
1. 接警后立即出警，现场控制局面
2. 调取现场监控
3. 询问当事人及目击证人
4. 调取医院诊断证明
5. 双方同意调解

**处理结果：**
经调解，张三向李四赔礼道歉，赔偿医疗费等损失共计800元，双方达成和解协议。公安机关不予行政处罚。

**办案要点：**
- 及时出警控制现场
- 固定证据（监控、证人）
- 医疗诊断是关键证据
- 民间纠纷引起的轻微案件可调解

---

### 案例2：酒后殴打他人

**基本案情：**
王五酒后因琐事与赵六发生争执，王五用啤酒瓶砸向赵六，致赵六头部受伤流血。经鉴定，赵六头部损伤构成轻微伤。

**调查情况：**
1. 现场勘查，提取作案工具（啤酒瓶碎片）
2. 调取酒吧监控
3. 询问当事人及目击证人
4. 法医鉴定意见
5. 王五酒醒后制作笔录

**处理结果：**
王五殴打他人，情节较重，根据《治安管理处罚法》第四十三条第一款，决定给予王五行政拘留十日，并处五百元罚款。

**办案要点：**
- 酒后作案不从轻
- 伤情鉴定是关键
- 使用工具从重处罚
- 监控录像重要证据

## 二、盗窃案例

### 案例3：超市盗窃案

**基本案情：**
孙七在某超市购物时，将价值约300元的商品藏匿于随身包中，未结账即离开超市。被超市保安发现并报警。

**调查情况：**
1. 超市监控录像清晰记录作案过程
2. 被盗商品已追回
3. 孙七承认违法事实
4. 孙七无前科

**处理结果：**
孙七盗窃，情节较轻，根据《治安管理处罚法》第四十九条，决定给予孙七行政拘留五日，并处五百元罚款。

**办案要点：**
- 监控录像直接证据
- 违法数额虽小但已构成违法
- 承认事实从轻处理
- 追回赃物酌情从轻

---

### 案例4：多次盗窃电动车电瓶

**基本案情：**
周八于2025年1-2月期间，先后4次在小区内盗窃电动车电瓶，共计价值约1500元。案发后被抓获。

**调查情况：**
1. 多名受害人报案
2. 小区监控录像
3. 销赃地点查获部分电瓶
4. 周八供述
5. 部分电瓶已发还受害人

**处理结果：**
周八盗窃，情节较重，根据《治安管理处罚法》第四十九条，决定给予周八行政拘留十五日，并处一千元罚款。

**办案要点：**
- 多次盗窃属情节较重
- 视频串并案
- 追缴赃物
- 累犯从重处罚

---

## 三、故意损毁财物案例

### 案例5：酒后砸车案

**基本案情：**
吴九酒后因找不到回家的路，对停放在路边的两辆轿车进行踢踹，致车辆多处凹陷、划痕。经鉴定，损失共计约2000元。

**调查情况：**
1. 接警后现场勘查
2. 调取附近监控
3. 车损鉴定
4. 吴九酒醒后承认事实
5. 联系车主进行赔偿

**处理结果：**
吴九故意损毁财物，根据《治安管理处罚法》第四十九条，决定给予吴九行政拘留十日，并处五百元罚款。

**办案要点：**
- 酒后滋事常见
- 车损鉴定确定损失数额
- 积极赔偿可酌情从轻
- 赔偿后减轻处罚

## 四、扰乱公共秩序案例

### 案例6：公共场所起哄闹事案

**基本案情：**
沈十一在某商场购物时，因不满服务态度，在大厅大声喧哗、拍打柜台，造成大量顾客围观，商场正常经营秩序受影响约半小时。

**调查情况：**
1. 商场监控录像
2. 商场工作人员陈述
3. 现场顾客证言
4. 沈十一承认违法事实

**处理结果：**
沈十一扰乱公共场所秩序，根据《治安管理处罚法》第二十三条第一款第二项，决定给予沈十一警告，并处二百元罚款。

**办案要点：**
- 扰乱秩序的认定
- 影响程度评估
- 首次违法可从轻
- 与寻衅滋事罪区分

---

### 案例7：散布谣言案

**基本案情：**
郑十二在某微信群发布"本地发生疫情，已有多人感染"的虚假信息，被大量转发，造成一定社会恐慌。经核查，该信息不实。

**调查情况：**
1. 网络巡查发现线索
2. 固定聊天记录
3. 核实信息真实性
4. 传唤郑十二调查
5. 郑十二承认编造传播

**处理结果：**
郑十二散布谣言扰乱公共秩序，根据《治安管理处罚法》第二十五条第一项，决定给予郑十二行政拘留五日，并处五百元罚款。

**办案要点：**
- 网络谣言及时处置
- 电子证据固定
- 查清信息源头
- 防止信息扩散
- 与诽谤罪区分

## 五、妨害公务案例

### 案例8：阻碍民警执法案

**基本案情：**
民警在现场处置纠纷时，王十三对民警进行辱骂、推搡，并试图抢夺民警执法记录仪，阻碍民警正常执法。

**调查情况：**
1. 执法记录仪录像
2. 现场其他证人证言
3. 民警受伤照片
4. 王十三供述

**处理结果：**
王十三阻碍执行职务，情节严重，根据《治安管理处罚法》第五十条第一款第二项、第二款，决定给予王十三行政拘留十日。

**办案要点：**
- 执法记录仪关键证据
- 暴力程度影响处罚
- 妨害公务从严处理
- 民警权益保护

## 六、赌博案例

### 案例9：聚众赌博案

**基本案情：**
民警接群众举报，在某民房内查获以麻将方式进行赌博的冯十四等8人，当场缴获赌资共计3万余元及赌具麻将机2台。

**调查情况：**
1. 现场检查笔录及照片
2. 赌资清点登记
3. 参赌人员询问笔录
4. 房东询问笔录
5. 冯十四承认组织赌博

**处理结果：**
- 冯十四为赌博提供条件，情节严重，行政拘留十二日，并处二千元罚款
- 其他参赌人员，每人行政拘留五日，并处五百元罚款

**办案要点：**
- 现场控制与取证
- 赌资认定与清点
- 组织者与参与者区分
- 赌具没收

## 七、办案经验总结

### 7.1 行政案件特点

1. **多发常见**：殴打、盗窃、损毁财物等
2. **可调解性**：民间纠纷引起的可调解
3. **程序相对简便**：办理期限较短
4. **注重教育**：处罚与教育相结合

### 7.2 证据要点

1. **及时性**：第一时间固定证据
2. **监控录像**：重要证据来源
3. **证人证言**：佐证作用
4. **违法人陈述**：认错态度

### 7.3 调解适用

**可调解情形：**
- 民间纠纷引起
- 情节较轻
- 双方自愿
- 非严重违法行为

**调解要点：**
- 自愿原则
- 合法原则
- 及时履行
- 制作协议

### 7.4 与刑事案件衔接

**需转刑事的情形：**
- 伤情达到轻伤以上
- 盗窃数额达到刑事标准
- 多次盗窃、入户盗窃
- 情节严重、社会危害大

**衔接程序：**
- 行政调查中发现涉嫌犯罪
- 制作《移送案件通知书》
- 移送刑侦部门
- 行政处罚可折抵刑罚
