# 程序操作指引

## 一、接处警操作要点

### 1.1 接警登记

**必录信息：**
- 报警时间（精确到分）
- 报警人姓名、电话
- 发案地点（详细地址）
- 案情简要（时间、人物、事件）
- 警情类型（刑事/行政/求助）
- 紧急程度

**注意事项：**
- 边接听边录入
- 关键要素问清楚
- 特殊情况做标记
- 敏感警情及时上报

### 1.2 出警要求

**时间要求：**
- 重大警情：立即
- 紧急警情：5-10分钟
- 一般警情：15-30分钟

**装备准备：**
- 执法记录仪（必开）
- 警械（根据情况）
- 相关法律文书
- 通讯设备

### 1.3 现场处置

**第一步：控制现场**
- 隔离围观人员
- 控制嫌疑人员
- 保护现场证据

**第二步：初步了解**
- 询问报警人
- 询问目击者
- 观察现场情况

**第三步：判断处理**
- 行政案件自行处理
- 刑事案件移交刑侦
- 需支援的及时请求

**第四步：固定证据**
- 拍照录像
- 提取痕迹物证
- 登记证人信息

## 二、受立案操作

### 2.1 受案程序

```
接报 → 制作受案登记表 → 出具受案回执 → 初步审查
```

**必做事项：**
- 当场出具受案回执
- 告知报案人权利义务
- 初步判断案件性质
- 决定是否立案

### 2.2 立案审查

**审查内容：**
- 是否有犯罪事实
- 是否需要追究刑事责任
- 是否属于管辖范围
- 是否超过追诉时效

**审查时限：**
- 一般案件：7日内
- 重大复杂：30日内
- 信访举报等复杂：60日内

### 2.3 立案或不立案决定

**立案：**
- 制作《立案决定书》
- 通知报案人、控告人
- 开始侦查程序

**不立案：**
- 制作《不予立案通知书》
- 说明理由
- 告知复议、复核权利

## 三、强制措施操作

### 3.1 传唤

**程序：**
- 一般需传唤证
- 紧急情况可先口头后补
- 告知被传唤人权利义务
- 限时8小时，复杂不超过24小时

**注意事项：**
- 不得连续传唤变相拘禁
- 不得异地传唤（特殊除外）
- 保障饮食休息

### 3.2 拘留

**提请批捕时限：**
- 一般：3日内
- 特殊：延长1-4日
- 流窜、结伙：延长至30日

**执行拘留：**
- 出示拘留证
- 24小时内讯问
- 24小时内通知家属
- 无法通知的记录在案

### 3.3 逮捕

**提请批准：**
- 拘留后3日内提请
- 检察院7日内决定
- 执行逮捕后24小时内讯问

**执行逮捕：**
- 出示逮捕证
- 通知家属
- 送看守所羁押

### 3.4 取保候审

**申请：**
- 犯罪嫌疑人及其法定代理人申请
- 公安机关3日内作出决定

**保证方式：**
- 保证金：起点1000元
- 保证人：符合条件的人

**监管：**
- 定期报到
- 限制离开居住地
- 禁止接触特定人员

## 四、侦查取证操作

### 4.1 讯问犯罪嫌疑人

**准备工作：**
- 了解案情
- 制定讯问提纲
- 准备录音录像设备

**讯问流程：**
1. 核实身份
2. 告知权利义务
3. 询问基本情况
4. 询问案件事实
5. 核对笔录
6. 签名捺印

**注意事项：**
- 全程同步录音录像
- 严禁刑讯逼供
- 允许辩解
- 笔录如实记录

### 4.2 询问证人

**程序要求：**
- 个别进行
- 告知如实作证义务
- 告知伪证法律责任
- 询问笔录签字

**特殊证人：**
- 未成年人：通知法定代理人
- 被害人：注意心理状态
- 伤残人员：提供便利

### 4.3 现场勘查

**组织指挥：**
- 指定勘查指挥员
- 分工明确
- 佩戴装备

**勘查流程：**
1. 保护现场
2. 观察巡视
3. 拍照录像
4. 绘制现场图
5. 提取痕迹物证
6. 制作勘查笔录

**注意事项：**
- 见证人在场
- 原物原貌记录
- 提取程序规范
- 手续齐全

### 4.4 搜查

**程序：**
- 持搜查证（紧急情况除外）
- 有见证人在场
- 制作搜查笔录
- 开具扣押清单

**注意事项：**
- 不得无故搜查
- 不得泄露隐私
- 女性由女工作人员检查
- 及时退还与案件无关物品

### 4.5 扣押

**范围：**
- 与案件有关的物品
- 可用以证明犯罪的物品

**程序：**
- 会同在场见证人
- 当场清点
- 开具扣押清单
- 通知领回（与案件无关的）

**保管：**
- 妥善保管
- 随案移送
- 不得挪用

## 五、文书制作操作

### 5.1 笔录制作要点

**基本要素：**
- 时间、地点
- 人员信息
- 权利告知
- 问答内容
- 核对确认
- 签名捺印

**注意事项：**
- 如实记录
- 字迹清晰
- 不得修改（修改需签名）
- 完整无遗漏

### 5.2 决定书制作

**结构：**
- 首部：当事人信息
- 正文：事实、证据、法律依据
- 尾部：决定内容、救济途径、日期盖章

**要求：**
- 事实清楚
- 证据充分
- 引用法律准确
- 格式规范

### 5.3 送达程序

**方式：**
- 直接送达（首选）
- 留置送达（拒绝签收）
- 邮寄送达
- 公告送达（最后选择）

**要求：**
- 送达回证签字
- 注明送达日期
- 受送达人签名

## 六、调解处理操作

### 6.1 调解适用

**可调解案件：**
- 民间纠纷引起
- 情节较轻
- 双方自愿

**不适用调解：**
- 雇凶伤害
- 涉黑涉恶
- 多次违反治安管理
- 当事人明确拒绝

### 6.2 调解程序

```
申请 → 受理 → 调查 → 组织调解 → 达成协议 → 履行 → 结案
```

**注意事项：**
- 自愿原则
- 合法原则
- 不得强迫
- 协议可执行

### 6.3 调解不成

- 制作调解终结书
- 告知可就民事争议提起民事诉讼
- 依法给予行政处罚

## 七、常见时限

| 程序 | 时限 |
|------|------|
| 传唤 | 8小时/24小时 |
| 拘留后提请批捕 | 3日/7日/30日 |
| 检察院批捕 | 7日 |
| 逮捕后侦查羁押 | 2个月（可延长） |
| 审查起诉 | 1个月（可延长15日） |
| 拘留后通知家属 | 24小时 |
| 逮捕后通知家属 | 24小时 |
| 取保候审 | 12个月 |
| 监视居住 | 6个月 |

## 八、执法记录仪使用

### 8.1 必须使用场景

- 接处警
- 现场勘查
- 询问讯问
- 强制措施
- 调解处理

### 8.2 使用要求

- 执法前开启
- 全程记录
- 结束后确认
- 及时上传保存

### 8.3 注意事项

- 告知当事人正在录音录像
- 记录内容完整
- 不得中断（特殊情况除外）
- 妥善保管
