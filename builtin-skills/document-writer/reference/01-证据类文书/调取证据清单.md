# 调取证据清单

## 文书说明

调取证据清单用于向有关单位或个人调取与案件有关的证据时使用。

## 填写说明

### 证据持有人/单位

**个人：**
- 姓名、身份证号
- 联系电话、住址

**单位：**
- 单位全称
- 单位地址
- 联系人、职务、电话

### 常见调取证据类型

| 证据类型 | 说明 | 备注 |
|---------|------|------|
| 监控录像 | 监控视频、行车记录仪 | 注明时间范围 |
| 通信记录 | 通话清单、短信记录 | 运营商提供 |
| 交易记录 | 银行流水、转账记录 | 银行提供 |
| 登记信息 | 住宿、上网、购票 | 相关单位提供 |
| 文书材料 | 合同、票据、证件 | 原件或复印件 |
| 电子数据 | 聊天记录、邮件 | 提取固定 |

### 特征描述填写要点

**监控录像：**
- 时间范围、地点
- 设备编号、通道
- 存储介质

**通信记录：**
- 通话双方号码
- 通话时间、时长
- 基站位置

**交易记录：**
- 交易时间
- 交易金额
- 账户信息
