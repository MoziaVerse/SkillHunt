# 随案移送物品清单

## 文书说明

随案移送物品清单用于案件移送（如移送检察院审查起诉）时，随案移送相关物品的清单。

## 填写说明

### 移送物品范围

**应当移送的物品：**
- 作为证据的实物
- 违法所得
- 非法财物
- 被害人合法财物（待发还）

**不宜移送的物品：**
- 危险品
- 大型物品
- 鲜活易腐物品
- 保密物品
- 其他特殊物品

### 不移送原因说明

| 原因类型 | 说明 | 处理方式 |
|---------|------|---------|
| 危险品 | 枪支弹药、爆炸物等 | 专用场所保管 |
| 大型物品 | 车辆、大型设备等 | 拍照固定，清单移送 |
- 鲜活易腐 | 食品、药品等 | 拍照固定后处理 |
| 保密物品 | 涉密文件资料 | 按保密规定处理 |
| 其他 | 需专门保管 | 另行处理 |

## 注意事项

1. **核对一致**：清单物品与实物一致
2. **封装规范**：物品妥善包装
3. **随案移送**：与案卷材料一并移送
4. **办理交接**：双方经办人交接签字
5. **保留存根**：移送单位留存复印件
