# 发还物品清单

## 文书说明

发还物品清单用于发还与案件无关的物品、被害人合法财物或已结案应发还的物品。发还时应当由领取人当面清点确认。

## 填写说明

### 案件信息
- **案件编号**：与原扣押清单一致
- **案件名称**：与原扣押清单一致

### 领取人信息
- **姓名**：领取人真实姓名
- **与案件关系**：被害人、嫌疑人、物品持有人等
- **身份证号**：18位身份证号码

### 发还原因勾选

根据实际情况勾选或填写：

| 情形 | 说明 |
|------|------|
| 与案件无关 | 扣押后经审查与案件无关 |
| 案件办结 | 案件已处理完毕应发还 |
| 被害人财物 | 被害人合法财产应及时发还 |
| 不起诉 | 检察院决定不起诉 |
| 判决无罪 | 法院判决无罪 |
| 其他原因 | 特殊情况 |

### 物品填写要点

| 项目 | 要求 |
|------|------|
| 物品名称 | 与原扣押清单一致 |
| 数量 | 与原扣押清单一致 |
| 特征描述 | 与原扣押清单一致 |
| 价值 | 与原扣押清单一致 |
| 备注 | 填写"已发还" |

## 发还程序

### 1. 审核发还条件

- 确认与案件无关或应发还
- 案件是否已办结
- 是否有其他机关需要

### 2. 核对物品

- 与扣押清单核对
- 确认数量、特征一致
- 检查是否有损坏

### 3. 通知领取人

- 电话通知
- 领取时需携带身份证
- 委托他人领取需出具委托书

### 4. 办理发还

- 当场清点
- 签名确认
- 收回原扣押清单
- 制作发还清单

## 特殊情形处理

### 领取人不能亲自领取

```
委托领取声明

本人__________（身份证号：__________________）
因____________________原因，不能亲自领取被
扣押物品，现委托__________（身份证号：
__________________）代为领取，所有法律后果
由本人承担。

委托人（签名）：_____________
日期：____年__月__日

受托人（签名）：_____________
身份证号：__________________
联系电话：__________________
日期：____年__月__日
```

### 物品损坏或短缺

- 在清单备注中注明
- 领取人声明中填写
- 视情况处理或赔偿

### 无主物品

- 公告认领
- 公告期满无人认领的
- 按规定处理

## 注意事项

1. **核对一致**：发还物品应与扣押清单一致
2. **当场清点**：领取人必须当场清点确认
3. **收回原单**：发还时收回原扣押清单
4. **签字确认**：领取人必须亲笔签名
5. **保存记录**：发还清单附卷保存
