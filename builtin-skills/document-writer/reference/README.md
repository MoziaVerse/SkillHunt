# 执法文书模板库

## 文档库说明

本模板库为公安执法文书制作提供标准化的文书模板和填写指引，涵盖案件办理各环节的常用文书。

## 📁 目录结构

### 01-证据类文书
与案件证据相关的文书模板

- **扣押物品清单** - 扣押与案件有关的物品
- **发还物品清单** - 发还与案件无关或已结案的物品
- **调取证据清单** - 向有关单位或个人调取证据
- **随案移送物品清单** - 案件移送时随案移送物品

### 02-程序类文书
案件办理程序相关的文书模板

- **受案回执** - 受理案件后出具给报案人
- **传唤证** - 传唤嫌疑人到案
- **搜查证** - 对人身、场所进行搜查
- **检查证** - 检查人身、物品、场所

### 03-决定类文书
各类决定书模板

- **行政处罚决定书** - 对违法行为作出处罚
- **不予处罚决定书** - 不予行政处罚

### 04-其他文书
其他常用文书模板

- **鉴定委托书** - 委托鉴定机构进行鉴定
- **协助查询通知书** - 查询存款、汇款等
- **送达回证** - 证明文书已送达

## 📋 填写要点

### 通用要求

| 要求 | 说明 |
|------|------|
| 字迹 | 使用黑色签字笔或打印 |
| 清晰 | 字迹工整，不得潦草 |
| 涂改 | 不得涂改，涂改处需签名或盖章 |
| 签名 | 日期、签名必须手写 |
| 编号 | 按单位规则编号，保持连续 |

### 关键要素

- **案件信息**：编号、名称、时间、地点
- **人员信息**：姓名、身份证号、联系方式
- **物品信息**：名称、数量、特征、价值、来源
- **程序信息**：法律依据、权利告知、救济途径
- **签名区域**：当事人、办案人、见证人、审批人
