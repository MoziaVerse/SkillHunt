# 检查证

## 文书说明

检查证是公安机关对人身、物品、场所进行检查时使用的法律文书，主要用于行政案件或特定检查。

## 模板格式

```

              检 查 证
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

X公（__）检字〔____〕第____号

______公安局
```

## 填写说明

### 文书编号

格式：`地区简称+部门简称+检字+年份+编号`

**示例：**
- X公（治）检字〔2025〕第00123号

### 检查类型

| 检查类型 | 法律依据 | 审批要求 |
|---------|---------|---------|
| 人身检查 | 治安管理处罚法87条 | 派出所/公安分局批准 |
| 住所检查 | 治安管理处罚法87条 | 县级以上公安机关 |
| 场所检查 | 治安管理处罚法87条 | 根据情况 |
| 物品检查 | 治安管理处罚法87条 | 一般情况 |

### 检查目的填写

**示例：**
```
检查是否有赌博用的麻将桌、筹码等物品，
检查是否有赌博人员。
```

## 检查与搜查的区别

| 项目 | 检查 | 搜查 |
|------|------|------|
| 适用案件 | 行政案件为主 | 刑事案件 |
| 法律依据 | 治安管理处罚法 | 刑事诉讼法 |
| 审批级别 | 住所检查需县级以上 | 一般为公安机关 |
| 强制程度 | 相对较弱 | 较强 |
| 紧急无证 | 不可以 | 可以（逮捕拘留时） |

## 注意事项

1. **出示证件**：检查证和人民警察证
2. **见证在场**：应有见证人或被检查人在场
3. **住所检查**：需县级以上公安机关开具检查证
4. **女性检查**：检查妇女身体由女工作人员
5. **制作笔录**：检查情况制作笔录
6. **文明检查**：不得无故损坏物品
