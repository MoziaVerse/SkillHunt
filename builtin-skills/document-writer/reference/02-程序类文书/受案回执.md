# 受案回执

## 文书说明

受案回执是公安机关受理案件后出具给报案人、控告人、举报人的凭证，证明公安机关已受理该案。

## 填写说明

### 报案人信息

| 项目 | 填写要求 |
|------|---------|
| 称谓 | 根据身份选择先生/女士/单位 |
| 报案时间 | 精确到分钟 |
| 案情简述 | 简明描述，如"手机被盗" |

### 受案编号

格式：`地区简称+部门简称+受字+年份+编号`

**示例：**
- 京公（朝）受字〔2025〕第00123号
- 沪公（浦）受字〔2025〕第00456号

### 案件性质

根据实际情况勾选：
- **刑事案件**：涉嫌犯罪，需追究刑事责任
- **行政案件**：违反治安管理，需行政处罚
- **其他**：非公安机关处理范围

### 查询信息

- **联系电话**：办案单位值班电话或民警手机
- **联系人**：主办民警姓名
- **查询时间**：工作时间

## 报案人分类

### 1. 被害人报案

当事人直接报案，回执抬头填写被害人姓名。

### 2. 目击者举报

目击违法行为，回执抬头填写举报人姓名（可匿名）。

### 3. 单位报案

单位财物受损或员工犯罪，回执抬头填写单位名称。

### 4. 代人报案

受委托代为报案，回执抬头填写实际被害人姓名。

## 权利告知说明

### 补充证据

报案人可补充提供：
- 现场照片/视频
- 涉案物品
- 证人联系方式
- 其他相关材料

### 办理进展通知

公安机关将依法通知：
- 立案或不立案决定
- 嫌疑人处理情况
- 物品发还通知
- 案件办结情况

### 救济途径

**刑事案件不予立案：**
- 向作出决定的公安机关申请复议
- 向同级人民检察院申请立案监督
- 向人民法院提起自诉

**行政案件不予处理：**
- 申请行政复议
- 提起行政诉讼

## 送达方式

| 方式 | 说明 |
|------|------|
| 当场领取 | 报案人亲自领取 |
| 邮寄送达 | 通过邮政快递寄送 |
| 电子送达 | 通过短信、APP推送 |

## 注意事项

1. **当场出具**：受案后应当场出具回执
2. **编号唯一**：每案一号，不得重复
3. **信息准确**：联系电话等需准确有效
4. **妥善保管**：报案人需保管好回执
5. **变更通知**：联系方式变更需及时告知
