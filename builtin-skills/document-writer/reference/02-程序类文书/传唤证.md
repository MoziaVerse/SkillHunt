# 传唤证

## 文书说明

传唤证是公安机关传唤嫌疑人到案接受讯问的法律文书。除紧急情况外，传唤需出示传唤证。

## 模板格式

```

              传 唤 证
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

X公（__）传字〔____〕第____号

______公安局

被传唤人：

性别：____ 出生日期：____年__月__日
身份证件种类：__________号码：__________________
住址：________________________________________

传唤理由：____________________________________
_______________________________________________

传唤地点：____________________________________
传唤时间：____年__月__日 __时

法律依据：
根据《中华人民共和国刑事诉讼法》第一百一十九条/
《中华人民共和国治安管理处罚法》第八十二条之规定，
特向你传唤。

权利告知：
1. 你有权委托律师作为辩护人/代理人。
2. 你有权对与本案无关的问题拒绝回答。
3. 你有权要求回避。
4. 你有权核对讯问笔录，确认无误后签名或盖章。

注意事项：
1. 请按时到达传唤地点，无正当理由不得拒绝。
2. 传唤时间不得超过12小时，案情重大、复杂不得超过24小时。
3. 请携带本人身份证件。

```

## 填写说明

### 文书编号

格式：`地区简称+部门简称+传字+年份+编号`

**示例：**
- X公（刑）传字〔2025〕第00123号
- X公（治）传字〔2025〕第00456号

### 被传唤人信息

| 项目 | 填写要求 |
|------|---------|
| 姓名 | 身份证件上的姓名 |
| 性别 | 男/女 |
| 出生日期 | 身份证上的出生日期 |
| 身份证件 | 身份证/护照等 |
| 住址 | 户籍登记或现居住地址 |

### 传唤理由

简明扼要说明传唤原因：

**刑事案件示例：**
- "因涉嫌盗窃罪"
- "因涉嫌故意伤害罪"
- "因涉嫌诈骗罪"

**行政案件示例：**
- "因殴打他人"
- "因盗窃"
- "因扰乱公共秩序"

### 传唤地点

| 类型 | 地点 |
|------|------|
| 常规 | XX派出所/公安局讯问室 |
| 特殊 | 嫌疑人住处/单位（需批准） |
| 异地 | 协作地公安机关 |

## 传唤时限

| 情形 | 时限 |
|------|------|
| 一般情况 | 12小时 |
| 案情重大、复杂 | 24小时 |
| 不得以连续传唤方式变相拘禁 | |

## 传唤证使用

### 必须使用传唤证的情形

- 传唤未羁押的嫌疑人到案
- 传唤需要讯问查证的人
- 传唤拒不到案的人

### 可以口头传唤的情形

- 现场发现的犯罪嫌疑人
- 扭送的犯罪嫌疑人
- 投案自首的犯罪嫌疑人

**注意：**口头传唤后应补办传唤证。

## 送达程序

### 1. 宣告送达

- 当面宣告被传唤人
- 说明传唤事由和时间地点
- 告知权利义务

### 2. 签收确认

- 被传唤人签名确认
- 注明收到时间
- 拒绝签收的，送达人注明

### 3. 记录在案

- 送达过程记录
- 被传唤人配合情况
- 特殊情况说明

## 权利告知详解

### 委托辩护人/代理人

- 有权委托律师
- 经济困难可申请法律援助
- 未成年人应通知法定代理人

### 拒绝回答权

- 只对与本案无关的问题
- 不得以沉默抗拒侦查
- 身份情况必须回答

### 回避权

- 办案人员是当事人近亲属
- 办案人员与案件有利害关系
- 办案人员与当事人有其他关系

## 无正当理由拒绝传唤的处理

### 刑事案件

- 可以强制传唤
- 使用警械具约束
- 必要时使用武力

### 行政案件

- 可以强制传唤
- 使用警械具约束
- 不得过度使用武力

## 注意事项

1. **出示证件**：执行传唤应出示人民警察证
2. **文明执法**：语言文明，行为规范
3. **限时讯问**：不得超过传唤时限
4. **保障权利**：告知并保障各项权利
5. **同步录音**：讯问过程全程录音录像
