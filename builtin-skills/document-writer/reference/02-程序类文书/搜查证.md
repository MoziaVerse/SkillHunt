# 搜查证

## 文书说明

搜查证是公安机关对人身、场所、物品进行搜查时使用的法律文书。除紧急情况外，搜查需出示搜查证。

## 填写说明

### 文书编号

格式：`地区简称+部门简称+搜字+年份+编号`

**示例：**
- X公（刑）搜字〔2025〕第00123号

### 搜查对象勾选

| 对象类型 | 说明 | 示例 |
|---------|------|------|
| 人身 | 嫌疑人身体 | 搜查犯罪嫌疑人身体 |
| 住所 | 住宅、宿舍等 | 搜查犯罪嫌疑人住所 |
| 其他场所 | 办公室、仓库等 | 搜查犯罪嫌疑人办公室 |
| 物品 | 车辆、箱包等 | 搜查涉案车辆 |
| 电子数据 | 电脑、手机等 | 搜查电子设备 |

### 搜查目的填写

**示例：**
```
查找涉嫌盗窃罪的赃物（手机、现金等），
查找作案工具（手套、手电筒等），
查找与犯罪有关的其他物品。
```

### 搜查范围填写

**示例：**
```
XX市XX区XX路XX号XX室张三的住所，
包括客厅、卧室、厨房、卫生间、阳台等
全部区域。
```

## 搜查程序

### 1. 审批

- 主办民警提出申请
- 部门负责人审批
- 重大案件需分管局长审批

### 2. 执行

- 出示搜查证
- 通知见证人到场
- 有被搜查人/家属在场

### 3. 实施

- 控制现场
- 仔细搜查
- 发现可疑物品扣押
- 制作搜查笔录

### 4. 交接

- 开具扣押清单
- 当场清点
- 双方签字确认

## 无证搜查情形

**可以无证搜查：**
- 执行拘留、逮捕时遇到紧急情况
- 带领指认现场时发现可疑证据
- 抓捕现行犯时

**要求：**
- 紧急情况消失后应及时补办手续
- 在搜查笔录中说明情况

## 注意事项

1. **出示证件**：必须出示搜查证和人民警察证
2. **见证在场**：应有见证人在场
3. **女搜女性**：搜查妇女身体由女工作人员
4. **保护隐私**：注意保护被搜查人隐私
5. **及时扣押**：发现涉案物品及时扣押
6. **制作笔录**：搜查情况制作笔录
7. **录音录像**：重大案件应全程录音录像
