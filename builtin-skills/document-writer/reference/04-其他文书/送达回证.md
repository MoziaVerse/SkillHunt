# 送达回证

## 文书说明

送达回证用于证明法律文书已经送达相关当事人，是法律文书生效的重要凭证。

## 填写说明

### 文书信息

| 项目 | 填写内容 |
|------|---------|
| 文书名称 | 被送达文书的全称 |
| 文书编号 | 被送达文书的编号 |
| 送达文书数量 | 实际送达的份数 |

### 受送达人类型

| 类型 | 说明 |
|------|------|
| 当事人本人 | 案件当事人 |
| 法定代表人 | 单位负责人 |
| 法定代理人 | 未成年人/精神病人的监护人 |
| 诉讼代理人 | 委托的律师或代理人 |
| 同住成年家属 | 当事人不在时的代收人 |

### 送达方式说明

| 方式 | 条件 | 要求 |
|------|------|------|
| 直接送达 | 受送达人签收 | 受送达人签名 |
| 留置送达 | 拒绝签收 | 见证人见证+说明 |
| 委托送达 | 距离较远 | 委托当地机关 |
| 邮寄送达 | 直接送达困难 | 挂号信/快递 |
| 电子送达 | 当事人同意 | 短信/APP/邮件 |
| 公告送达 | 下落不明 | 公告60日 |

### 留置送达程序

受送达人拒绝签收时：
1. 邀请见证人到场
2. 说明情况
3. 在送达回证上注明
4. 见证人签名
5. 文书留在受送达处所

### 邮寄送达要求

- 使用挂号信或快递
- 保留邮寄凭证
- 查询送达状态
- 将凭证附卷

### 公告送达要求

- 受送达人下落不明
- 公告期为60日
- 在报纸/网站公告
- 公告期满视为送达

## 注意事项

1. **及时送达**：作出决定后应及时送达
2. **送达凭证**：送达回证是重要凭证
3. **送达顺序**：按直接→留置→委托→邮寄→公告顺序
4. **保存证据**：送达过程相关材料要保存
5. **附卷归档**：送达回证要附卷归档
