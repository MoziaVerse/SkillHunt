# 鉴定委托书

## 文书说明

鉴定委托书是公安机关委托鉴定机构对专门性问题进行鉴定时使用的法律文书。

## 填写说明

### 文书编号

格式：`地区简称+部门简称+鉴委字+年份+编号`

**示例：**
- X公（刑）鉴委字〔2025〕第00123号

### 鉴定事项勾选

根据需要勾选相应鉴定类型：

| 鉴定类型 | 适用情况 |
|---------|---------|
| 法医病理鉴定 | 死因鉴定、死亡方式判断 |
| 法医临床鉴定 | 伤情鉴定、伤残等级 |
| 法医精神病鉴定 | 精神状态、责任能力 |
| 法医物证鉴定 | 血迹、精斑、毛发等 |
| 法医毒物鉴定 | 毒物、毒品检测 |
| 司法会计鉴定 | 财务、账目审计 |
| 痕迹鉴定 | 指纹、足迹、工具痕迹 |
| 文书鉴定 | 笔迹、印章、文书真伪 |
| 微量物证鉴定 | 纤维、油漆、土壤等 |
| 声像资料鉴定 | 录音、录像真伪 |
| 电子数据鉴定 | 电子数据提取分析 |
| 价格鉴定 | 物品价值评估 |

### 具体鉴定要求

根据案件情况填写，例如：

**法医临床鉴定：**
```
1. 对张三的损伤程度进行鉴定（依据《人体损伤程度鉴定标准》）。
2. 明确损伤形成机制。
3. 明确损伤与外力的因果关系。
```

**痕迹鉴定：**
```
1. 对现场提取的指纹与嫌疑人张三的指纹进行比对。
2. 确定现场指纹是否为张三所留。
```

**价格鉴定：**
```
1. 对被盗窃的手机一部进行价值鉴定。
2. 品牌型号：华为Mate60 Pro
3. 购买时间：2024年5月
4. 使用状况：九成新
```

### 送检材料清单

| 材料类型 | 示例 |
|---------|------|
| 生物检材 | 血迹、精斑、毛发、指甲等 |
| 现场物品 | 作案工具、痕迹载体 |
| 文书材料 | 病历、X光片、CT片、合同等 |
| 电子数据 | 手机、电脑、硬盘等 |
| 录音录像 | 监控录像、录音文件 |

## 鉴定期限

| 鉴定类型 | 一般期限 |
|---------|---------|
| 法医临床 | 15个工作日 |
| 法医病理 | 30个工作日 |
| 法医物证 | 15个工作日 |
| 痕迹鉴定 | 15个工作日 |
| 文书鉴定 | 15个工作日 |
| 价格鉴定 | 7个工作日 |

**疑难复杂可延长：**
- 一般延长不超过30个工作日
- 延长需告知委托机关

## 鉴定人要求

- 具有相关鉴定资质
- 与案件无利害关系
- 需要回避的应当回避

## 鉴定意见内容

合格的鉴定意见应包含：
1. 委托事项
2. 鉴定材料
3. 鉴定依据
4. 分析说明
5. 鉴定意见
6. 鉴定人签名
7. 鉴定机构盖章
8. 鉴定日期

## 补充鉴定

**需要补充鉴定的情形：**
- 发现新的鉴定材料
- 原鉴定有遗漏
- 原鉴定意见不明确

**重新鉴定的情形：**
- 鉴定程序违法
- 鉴定人应当回避而未回避
- 鉴定依据明显不足
- 鉴定意见与其他证据矛盾

## 注意事项

1. **材料齐全**：送检材料要齐全、规范
2. **要求明确**：鉴定要求要明确、具体
3. **时限合理**：鉴定期限要合理
4. **保持联系**：与鉴定机构保持联系
5. **意见审查**：收到鉴定意见后及时审查
