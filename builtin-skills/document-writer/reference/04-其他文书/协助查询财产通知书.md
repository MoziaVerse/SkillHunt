# 协助查询财产通知书

## 文书说明

协助查询财产通知书用于查询犯罪嫌疑人、违法嫌疑人的存款、汇款、证券、基金等财产信息。

## 填写说明

### 文书编号

格式：`地区简称+部门简称+协查字+年份+编号`

**示例：**
- X公（刑）协查字〔2025〕第00123号

### 查询事项勾选

根据需要勾选相应查询类型

### 查询时间范围

- 根据案件需要确定
- 一般为案发前后一段时间
- 重大案件可查询更长周期

## 注意事项

1. **双人执行**：需两名民警持证办理
2. **保守秘密**：查询结果要保密
3. **及时固定**：收到结果及时固定
4. **可以冻结**：查询后可立即冻结
5. **与冻结衔接**：查询与冻结通知书可同时使用
