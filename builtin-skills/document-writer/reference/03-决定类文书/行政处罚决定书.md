# 行政处罚决定书

## 文书说明

行政处罚决定书是公安机关对违反治安管理等违法行为人作出处罚决定的正式法律文书。

## 模板格式

## 填写说明

### 文书编号

格式：`地区简称+部门简称+行罚决字+年份+编号`

**示例：**
- X公（治）行罚决字〔2025〕第00123号

### 被处罚人信息

| 项目 | 填写要求 |
|------|---------|
| 姓名 | 身份证件上的姓名 |
| 出生日期 | 身份证上的出生日期 |
| 身份证号 | 18位身份证号码 |
| 户籍地址 | 户籍登记地址 |
| 现住址 | 现实际居住地址 |
| 工作单位 | 全称 |
| 联系电话 | 有效手机号码 |

### 违法事实

#### 时间要素
- 年、月、日、时、分要精确
- 例："2025年3月17日14时30分许"

#### 地点要素
- 具体到街道、门牌号
- 例："XX市XX区XX路XX号XX超市内"

#### 行为描述
- 客观描述行为过程
- 例："趁收款员不备，将收银台上的现金200元放入口袋后离开"

### 证据列举

按证据类型列举：

| 证据类型 | 说明 | 示例 |
|---------|------|------|
| 被处罚人陈述 | 承认违法事实的笔录 | 讯问笔录第X页 |
| 被害人陈述 | 受害人描述 | 询问笔录第X页 |
| 证人证言 | 目击者描述 | XX询问笔录 |
| 视听资料 | 监控、录音录像 | 监控录像光盘 |
| 书证 | 文件、字据等 | 转账记录 |
| 鉴定意见 | 专业鉴定 | 伤情鉴定书 |
| 勘验笔录 | 现场勘查 | 现场勘查笔录 |

### 情节认定

**从重情节：**
- 教唆、胁迫他人违法
- 对未成年人、老年人等实施违法行为
- 多次实施违法
- 结伙作案
- 拒绝、阻碍执法

**从轻情节：**
- 主动投案
- 如实陈述
- 配合调查
- 初次违法

**减轻情节：**
- 未成年人
- 主动消除或减轻违法后果
- 受他人胁迫
- 有立功表现

## 处罚种类

### 1. 警告

最轻的行政处罚，适用于情节轻微的行为。

### 2. 罚款

| 违法行为 | 罚款幅度 |
|---------|---------|
| 情节较轻 | 200-500元 |
| 情节一般 | 500-1000元 |
| 情节较重 | 1000-3000元 |
| 情节严重 | 3000-5000元 |

### 3. 行政拘留

| 违法行为 | 拘留期限 |
|---------|---------|
| 情节较轻 | 5日以下 |
| 情节一般 | 5-10日 |
| 情节较重 | 10-15日 |

### 4. 吊销许可证

吊销公安机关发放的许可证，如：
- 特种行业许可证
- 枪支管理许可证
- 保安服务许可证

### 5. 其他处罚

- 限期出境
- 驱逐出境（外国人）

## 履行方式

### 罚款缴纳

- 期限：15日内
- 地点：指定银行
- 方式：现金或转账
- 滞纳金：逾期每日加处罚款3%

### 行政拘留

- 地点：拘留所
- 携带物品：身份证、换洗衣物
- 禁止物品：现金、通讯工具等
- 家属通知：执行后24小时内通知

## 救济途径

### 行政复议

- 期限：收到决定书之日起60日
- 复议机关：上一级公安机关或本级人民政府
- 结果：维持、撤销、变更

### 行政诉讼

- 期限：直接起诉6个月，复议后起诉15日
- 管辖：行政机关所在地法院
- 结果：维持、撤销、变更

## 注意事项

1. **事实清楚**：违法事实要清楚、证据确实充分
2. **程序合法**：严格按法定程序办理
3. **适用准确**：引用法律条文要准确
4. **内容完整**：各项内容填写完整
5. **送达规范**：及时送达并告知权利
