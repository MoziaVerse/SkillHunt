# 不予行政处罚决定书

## 文书说明

不予行政处罚决定书是公安机关对违法行为不成立或依法不予处罚时使用的法律文书。

## 填写说明

### 文书编号

格式：`地区简称+部门简称+不罚决字+年份+编号`

**示例：**
- X公（治）不罚决字〔2025〕第00123号

### 不予处罚情形

| 情形 | 说明 | 法律依据 |
|------|------|---------|
| 事实不成立 | 无证据证明有违法行为 | 处罚法57条 |
| 行为轻微 | 及时改正，无危害后果 | 处罚法33条 |
| 无主观过错 | 证据证明无过错 | 处罚法33条 |
| 不满14周岁 | 未达到责任年龄 | 治安法12条 |
| 精神病人 | 不能辨认控制行为 | 治安法13条 |

### 教育措施

不予处罚不等于不管教，可采取：
- 批评教育
- 责令严加看管
- 责令监护人管教
- 责具结悔过
